# 🏪 Superstore Sales PowerBI Dashboard

## 📌 Project Overview
Sales and profitability analysis of a US retail store using 
Power BI and DAX. The dataset contains 9,994 records covering 
4 years of sales data across different categories, segments 
and regions.

## 📊 Dashboard Pages
- **Page 1 - Sales Summary:** KPIs, monthly trend, 
  top 10 products, category and segment analysis
- **Page 2 - Geographic & Regional Analysis:** 
  Sales performance by state and region  
- **Page 3 - Annual Performance:** 
  Year-over-year growth and trends

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (data transformation)

## 📁 Repository Structure
- `Database/` — Original dataset (CSV)
- `Dashboard/` — Power BI file (.pbix)

## 💡 Key DAX Measures
- Total Sales, Total Profit, Profit Margin %
- Total Orders, Total Quantity
- Average Sales per Order
